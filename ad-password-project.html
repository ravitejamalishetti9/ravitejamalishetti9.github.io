My Solution

  1. Deployed Windows Server 2022 as Domain Controller (AD-DC1) for company.local.
  2. Created OUs: HR, Finance, IT, and Workstations.
  3. Added users and groups (HR_Read, Finance_Full, IT_Admins).
  4. Configured shared folders with NTFS & share permissions (\\AD-DC1\HR-Shared).
  5. Applied GPOs to enforce password policies and desktop restrictions.
